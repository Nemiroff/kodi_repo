
KODI плагин для просмотра сериалов с LostFilm.TV
=====================================================
  
Просматривайте ваши любимые сериалы в онлайне, без предварительной загрузки.


Возможности плагина
-------------------

* **Просмотр сериалов в онлайне с возможностью перемотки**
* **Добавление любимых сериалов непосредственно в библиотеку Kodi**, с последующим получением информации об эпизодах с помощью инфоресурсов
* Поддержка ОС Android ARM, Linux x86/x64/ARM, Windows, Mac OS X
* Автоматическое сканирование сериалов из библиотеки на предмет выхода новых эпизодов
* Управление отметками просмотрено/не просмотрено
* Выбор качества видео (SD/HD720/HD1080) по умолчанию
* Выбор движка для просмотра: Torrent2http (встроенный), TorrServe, Elementum
* Загрузка эпизодов с помощью Transmission/uTorrent
* Сохранение файлов просмотренных эпизодов после просмотра (при желании)


Установка
---------

Установить плагин можно из моего [репозитория](https://nemiroff.github.io/kodi_repo/)


Часто задаваемые вопросы
------------------------

### Я добавил сериал в библиотеку, но он не появился в списке сериалов Kodi ###

* Возможно, вы еще не создали источник для сериалов. Плагин при первом запуске наверняка спрашивал вас об этом.
Если вы не сделали этого сразу, откройте настройки плагина и выберите "Создать источник для сериалов LostFilm...", 
после чего необходимо будет перезапустить Kodi.

* Убедитесь, что источник создан, для этого откройте раздел Видео > Файлы, источник плагина называется "LostFilm.TV"

* Возможно, инфоресурс thetvdb.com, устанавливаемый по умолчанию для сериалов, не знает русского наименования сериала.
Откройте источник Видео > Файлы > LostFilm.TV, в списке найдите добавленный сериал, вызовите контекстное меню и выберите 
пункт "Сведения о сериале". Если информация о сериале не найдена, Kodi предложит откорректировать название. Попробуйте
ввести английское название сериала.

### При открытии эпизода плагин пишет что сидов и пиров нет (S:0 P:0), загрузка не идет ###

* Возможно, необходимо просто подождать секунд 15-20

* Проверьте, открыт ли порт для входящих соединений (6881 по умолчанию, его можно изменить в настройках), это, в частности,
позволит улучшить поиск пиров

* Попробуйте выбрать другое качество видео (в настройках плагина)

### Просмотр часто приостанавливается, отображается окно с сообщением "Буферизация..." ###
 
* Возможно, ваше сетевое соединение слишком медленное, попробуйте выбрать в настройках другое качество просмотра

* Проверьте количестов сидов/пиров (нажмите на паузу во время просмотра), плагин отобразит информацию о загрузке. Если
количество сидов и пиров меньше 5, то попробуйте выбрать другое качество просмотра

### Что насчет раздачи? ###

* Раздача идет, пока вы смотрите эпизод и завершается по окончании просмотра.
 
### Сколько нужно ли свободного места на диске? ###
 
* Места необходимо не меньше, чем размер просматриваемого в данный момент файла. Просматриваемые файлы сохраняются во 
временную папку на диске и хранятся там до тех пор, пока они не станут в сумме занимать объем больше, чем указано в 
настройках (по умолчанию 5 Гб), после чего временная папка автоматически очищается. В случае использования TorrServe 
просматриваемый файл хранится по частям в оперативной памяти.
